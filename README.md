# AI Remote Exam Proctoring System

Full-stack AI-based remote proctoring platform built with:

- FastAPI (Backend)
- PostgreSQL
- YOLOv8 (Mobile Detection)
- MediaPipe (Face Detection)
- InsightFace (Face Verification)
- Electron (Secure Exam Client)
- Docker (Containerization)

## Features

- Face Enrollment & Verification
- Live Camera Monitoring
- Mobile Phone Detection
- Auto Termination on Violations
- Subject-wise Question Navigation
- Countdown Timer
- Result Storage (Admin Publish Later)
- Dockerized Microservice Architecture
